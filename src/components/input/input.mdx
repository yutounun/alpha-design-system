The **Input** is a styled native text field with `sm` / `md` / `lg` sizes, token-based borders and focus rings, and `aria-invalid` styling for validation.

## Usage

```tsx
import { Input } from "@/components/input"

export function Example() {
    return <Input placeholder="Search…" />
}
```
