The **Separator** is a thin rule for grouping content horizontally or vertically.

## Usage

```tsx
import { Separator } from "@/components/separator"

export function Example() {
    return (
        <div className="flex flex-col gap-3">
            <p>Above</p>
            <Separator />
            <p>Below</p>
        </div>
    )
}
```
